


# home
1、新建Repository名为xxx
   新建gh-pages分支


2、克隆到本地

>git clone https://github.com/user/xxx.git

3、创建并跟踪gh-pages分支

>git checkout -b gh-pages origin/gh-pages
     

4、同步到远程服务器

>git status

>git add .

>git commit -m "updated"

>git push origin gh-pages


5、通过[URL](https://user.github.io/xxx/)访问
